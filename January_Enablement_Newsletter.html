About the Bootcamps

Data Pioneer Bootcamp

Today’s businesses need more than just data—they need real-time insights that drive action. In this in-person bootcamp, you’ll get hands-on experience with Databricks AI/BI, learning how to seamlessly explore, analyze, and visualize your data with a Databricks Warehouse. With guidance from Databricks experts, you’ll see firsthand how an integrated approach to business intelligence can help you make faster, more informed decisions—all within a single, unified platform.

During this bootcamp, you will:
  • • Explore and organize your data using Unity Catalog
  • • Query and analyze your data with a Databricks Warehouse
  • • Build interactive AI/BI Dashboards to visualize key insights
  • • Go beyond dashboards with AI/BI Genie to interact in natural language
AI Pioneer Bootcamp

Unlock the power of AI with Databricks in this hands-on bootcamp that dives into GenAI, Advanced Machine Learning (ML), and MLOps/LLMOps. Equip yourself to build, deploy, and scale sophisticated AI models — all within the Databricks Platform. With expert guidance, you’ll explore cutting-edge AI techniques and learn how to integrate the entire AI lifecycle into your systems.

During this bootcamp, you will:
  • • Design robust, autonomous AI Agent Systems with the Agent Framework
  • • Use Agent Evaluation to optimize performance with custom metrics
  • • Streamline deployment via Batch Inference and Model Serving
  • • Utilize AI Gateway to simplify API management across models
  • • Maximize observability, security and governance with Unity Catalog
