  • • JSON pipeline architecture
  • • Data exploration with Databricks SQL

MACHINE LEARNING

End-to-End AI on Databricks

Tuesday, February 24, 2026 at 2:00 PM EST / 11:00 AM PT
Duration: 1.5 hours

Walk through an end-to-end AI agent solution on Databricks. Build AI Agents with foundation LLMs, RAG, Vector Search, and PDF extraction using Unity Catalog functions, LangChain, and Databricks' built-in ai_parse_document function. Deploy real-time Q&A chatbots, evaluate agents with Mosaic AI Agent Evaluation and MLflow 3.0, and monitor live production behavior.

AI agent development features:
  • • Build AI agents with foundation LLMs and RAG
  • • PDF extraction using ai_parse_document function
  • • Unity Catalog functions and Vector Search
  • • Deploy real-time Q&A chatbots with LangChain
  • • Evaluate with Mosaic AI Agent Evaluation and MLflow 3.0
  • • Monitor live agents and production behavior
Presenters:
Rupal Gupta (Sr. Solutions Engineer) and Bhagyarshri Badgujar (Solutions Architect)
